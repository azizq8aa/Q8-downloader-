# Q8 Downloader 🎬

موقع تحميل الفيديوهات من منصات التواصل الاجتماعي المختلفة

## 🌟 المميزات

- 📱 **5 منصات مدعومة**: YouTube, TikTok, Instagram, Twitter, Snapchat
- 🎵 **صيغتان**: MP4 (فيديو) و MP3 (صوت)
- 🌐 **واجهة عربية**: تصميم جميل ومتجاوب
- ⚡ **سريع وآمن**: تحميل مباشر بدون إعلانات
- 📱 **متوافق مع الجوال**: يعمل على جميع الأجهزة

## 🚀 الموقع المباشر

**[Q8 Downloader](http://178.18.252.4:5000)**

|---------|--------|-----------------|----------|

## 🛠️ التقنيات المستخدمة

### Backend
- **Python 3.12** - لغة البرمجة الأساسية
- **Flask** - إطار العمل الخلفي
- **yt-dlp** - مكتبة تحميل الفيديوهات
- **Flask-CORS** - دعم الطلبات من مصادر مختلفة

### Frontend  
- **HTML5** - هيكل الصفحة
- **CSS3** - التصميم والتنسيق
- **JavaScript** - التفاعل والوظائف
- **Cairo Font** - خط عربي جميل

### Infrastructure
- **Ubuntu Server** - نظام التشغيل
- **Nginx** - خادم الويب
- **Gunicorn** - خادم WSGI

## 📋 كيفية الاستخدام

1. **انسخ رابط الفيديو** من أي منصة مدعومة
2. **الصق الرابط** في الحقل المخصص
3. **اختر الصيغة** (MP4 أو MP3)
4. **اضغط تحليل الرابط** وانتظر
5. **حمل الملف** عند ظهور الرابط

## 🔌 APIs المتاحة

### POST /api/download
تحميل فيديو من رابط

**Request:**
```json
{
  "url": "https://www.youtube.com/watch?v=dQw4w9WgXcQ",
  "format": "mp4"
}
```

**Response:**
```json
{
  "success": true,
  "data": {
    "title": "عنوان الفيديو",
    "platform": "youtube",
    "format": "mp4",
    "download_url": "/api/file/abc123"
  }
}
```

### GET /api/file/{download_id}
تحميل الملف المحمل

### POST /api/info
الحصول على معلومات الفيديو بدون تحميل

### GET /api/supported-platforms
قائمة المنصات المدعومة

### GET /api/health
فحص حالة الخدمة

## ⚠️ ملاحظات مهمة

### YouTube و Instagram
هذه المنصات قد تطلب تحقق إضافي أحياناً، مما قد يؤدي إلى:
- رسائل "تحقق من أنك لست روبوت"
- حدود في عدد التحميلات
- الحاجة لانتظار فترة قبل المحاولة مرة أخرى

**الحلول البديلة:**
- استخدم TikTok أو Twitter كبديل
- انتظر قليلاً ثم أعد المحاولة
- تأكد أن المحتوى عام وليس خاص
